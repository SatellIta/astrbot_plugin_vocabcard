# AstrBot 每日单词卡片插件

> 每天定时推送精美的英语单词卡片，助力英语学习

## 一句话介绍

自动生成并推送玻璃拟态风格的英语单词卡片，内置 3722+ CET-6 词汇，支持定时推送和手动获取。

## 核心功能

- 🎨 **精美卡片**：玻璃拟态设计，超高清分辨率（1296×1620）
- 📅 **定时推送**：每天自动推送到群聊，时间可配置
- 📚 **丰富词库**：3722+ CET-6 核心词汇，随机/顺序学习
- 📊 **进度追踪**：自动记录学习进度，支持查看统计
- 🖼️ **多样背景**：37 张精选图片，随机裁剪，每天不重样

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt
playwright install chromium

# 注册推送
/vocab_register

# 手动获取
/vocab
```

## 适用场景

- 个人英语学习打卡
- 群聊每日单词推送
- 英语学习社群运营
- CET-6 备考辅助
